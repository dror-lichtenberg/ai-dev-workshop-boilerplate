## 📖 The Story: Why Do We Need GitHub?

### The Problem

Imagine you're writing a 50-page report with three teammates. You email the file back and forth:

```
report_v1.docx
report_v2_john_edits.docx
report_v3_FINAL.docx
report_v3_FINAL_REAL_FINAL.docx
report_v4_sarah_comments_IMPORTANT.docx
```

This is a **nightmare**. You can't tell who changed what. You can't undo mistakes. You might overwrite someone else's work.

### The Solution

**Git** was created in 2005 by Linus Torvalds (the same person who created Linux) to solve exactly this problem for software development.

**GitHub** (launched in 2008) puts Git on the internet so teams can collaborate.

> **Analogy: Git is like a time machine + a shared notebook.**
> - **Time machine**: You can go back to any point in your project's history
> - **Shared notebook**: Multiple people can write at the same time without overwriting each other
> - **Who changed what**: Every change is signed and dated

## 🔧 Steps

### Step 1: Create a GitHub Account

1. Go to [github.com](https://github.com)
2. Click **"Sign up"** (top right)
3. Enter your email address
4. Create a password
5. Choose a username (this will be your developer identity — pick something professional)
6. Complete the email verification (check your inbox for the code)
7. Complete the puzzle/verification

> **💡 Tip:** Your GitHub username is your professional identity. Many recruiters look at GitHub profiles. Choose something like `firstname.lastname` or `firstnamelastname`, not `xX_cool_coder_Xx`.

### Step 2: Verify Your Email

1. Check your email inbox for a message from GitHub
2. Click the verification link
3. Return to GitHub — you should see your dashboard

### Step 3: Explore the GitHub Interface

Look around your GitHub home page. Notice:
- **Repositories** — these are your projects (we'll create one soon)
- **Profile** — your public developer identity
- **Explore** — see what others are building

### Step 4: Activate Student Benefits (Optional but Recommended)

If you're a verified student:

1. Go to [education.github.com/pack](https://education.github.com/pack)
2. Click **"Get student benefits"**
3. Verify your student status (using your school email)
4. This gives you **free** access to:
   - GitHub Copilot (AI coding assistant) — normally $10/month
   - GitHub Codespaces (cloud development) — 180 hours/month
   - GitHub Pro features
   - 100+ other developer tools and services

> **⏱️ Note:** Student verification can take from a few minutes to a day. If it's not instant, don't worry — you can still participate in the workshop without it.
